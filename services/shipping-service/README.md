# Shipping Service

Microservicio Laravel para métodos de envío, reglas de costo y direcciones de usuario.

## Endpoints

- `GET /api/health`
- `GET /api/shipping/methods`
- `GET /api/shipping/rules`
- `POST /api/shipping/estimate`

## Tablas de dominio

- `shipping_methods`
- `shipping_price_rules`
- `user_addresses`
