# Audit Service

Microservicio Laravel para trazabilidad y auditoría de usuarios, productos, categorías y órdenes.

## Endpoints

- `GET /api/health`
- `GET /api/audits/orders`
- `GET /api/audits/users`
- `GET /api/audits/products`

## Tablas de dominio

- `audit_categories`
- `audit_orders`
- `audit_users`
- `productos_auditoria`
- `eliminaciones_auditoria`
